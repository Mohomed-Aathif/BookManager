# Book Manager Web Application

This project is a simple web application that allows users to manage a list of books.

## Technologies Used

Frontend:
Angular

Backend:
ASP.NET Core Web API with C#

## Features

Add a new book  
View list of books  
Update existing books  
Delete books  

Each book contains the following properties:
- Title
- Author
- ISBN
- Publication Date


## Project Structure

BookManagerAPI → ASP.NET Web API backend

book-manager-url → Angular frontend
